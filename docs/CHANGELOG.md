- 接入用户提供的网站 Logo 与默认头像素材，头像组件统一使用默认头像兜底。
- 参考“个人中心参考图”继续优化个人中心：保留左侧菜单、用户卡片、订单/收藏/优惠券/消息/售后/发票等卡片化入口。

### 第三轮：近期与中期功能补齐

- 补齐会员体系基础版：会员等级、积分到账、积分兑换优惠券、消息中心、登录通知、邮箱验证模拟流程、两步验证演示登录。
- 商品侧补齐规格/SKU 文本配置、下单规格记录、商品视频、限时活动价、活动标签、前台活动价展示。
- 搜索页新增评分筛选，并继续支持品牌、价格、库存、排序与 URL 状态保持。
- 首页新增“猜你喜欢”个性化/热销兜底推荐，商品卡片展示活动价与原价。
- 购物车、结算页、订单页支持商品规格展示；结算页运费与后端统一为满 99 包邮，否则 8 元。
- 结算页接入保存的发票抬头/税号，下单会写入订单发票信息。
- 新增售后申请入口、用户售后进度、后台售后审核，售后进度会通过消息中心通知。
- 商品评价新增追评入口和商家回复展示，后台评价管理支持回复评价。
- 后台优惠券管理新增优惠券模板、批量发券，发券后用户会收到消息通知。
- 后台用户管理补齐禁用原因、角色切换（管理员/运营/客服/普通用户），后端按角色限制后台接口。
- 安全侧补齐 JWT 退出黑名单、密码修改使当前 token 失效、文件上传文件头校验、登录/注册/下单/上传/评价基础限流。
- 数据库初始化补齐中期功能表与字段，包括消息、浏览历史、售后、优惠券模板、发票、商品规格/活动、购物车规格等。

### 第二轮：中期功能补充

- 商品增加品牌字段，支持用户端品牌搜索、后台品牌维护和品牌筛选。
- 搜索框与搜索页增加最近搜索记录，支持快速复搜和清空。
- 新增搜索关键词统计表与热门搜索/搜索建议接口，前台下拉和搜索页可动态展示热门词。
- 购物车增加失效商品提示，失效商品不可勾选结算。
- 结算页自动选择当前可用的最优优惠券，用户仍可手动切换或不使用。
- 商品评价支持多图链接展示，商品详情页和后台评价管理均可查看评价图片。

### 第一轮：近期可落地优化

本轮按“先做当前能落地的，复杂功能放后期”的原则，完成一批低风险优化。

### 用户端

- 订单增加状态筛选。
- 订单支持模拟支付、取消订单、确认收货。
- 订单卡片展示商品金额、优惠、运费、实付等价格明细。
- 结算页接入后端收货地址接口，并保留本地地址作为兜底。
- 个人中心新增收货地址管理：新增、编辑、删除、设为默认。
- 商品详情新增后端相关推荐接口调用，并保留同分类商品兜底推荐。
- 首页新增“最近浏览”模块。
- 搜索页支持把关键词、排序、价格区间、仅看有货、页码同步到 URL，刷新和分享链接时条件不会丢失。

### 管理端

- 商品管理新增批量上架、批量下架。
- 商品管理新增库存预警筛选。
- 商品管理新增当前页 CSV 导出。
- 商品创建/编辑增加后端基础校验：名称、分类、价格、库存、状态。
- 订单管理新增批量发货、批量完成、批量取消。
- 订单管理支持按订单号、收货人、手机号搜索。
- 订单详情弹窗补充商品明细和费用明细。
- 订单管理新增当前页 CSV 导出。
- 后台写操作新增审计日志，记录管理员、请求方法、路径和 IP。

### 后端与安全

- 新增收货地址表和 `/addresses` 接口。
- 新增商品相关推荐接口 `/products/{id}/recommendations`。
- 登录失败 5 次后锁定 10 分钟，降低暴力破解风险。
- 业务异常现在会尽量返回匹配的 HTTP 状态码，例如 429、403、404。
- 管理端取消订单时会归还库存，并恢复已使用优惠券。
- 后端新增收货地址单元测试。
